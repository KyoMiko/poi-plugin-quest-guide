# 简介

查询指定任务所有未完成前置的插件

# 安装
在poi浏览器中选择设置->扩展程序  
输入poi-plugin-quest-guide并选择安装  

# 使用介绍
本插件依赖于任务信息2插件，请确保已经安装任务信息2，重新启动poi后，请点击任务加载当前任务列表
插件输入框内输入任务编号，如A1，点击查询即可查询该任务的全部已知前置，以及其是否可接取、已完成

注意：本插件的前置信息来自于舰娘百科，且存在一定滞后性（舰c仍有大量任务前置未知），因此较新的任务可能查询不出任何结果
非单次任务（日周月季年）由于存在两种前置（本周是否完成及是否完成过），在判断前置时可能会存在本周期未完成但是分类在已完成任务中
